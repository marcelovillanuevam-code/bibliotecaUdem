# Artefactos de Trabajo  
## Sistema de Biblioteca Inteligente UDEM con Chatbot Integrado

Este repositorio contiene los artefactos generados durante el desarrollo del proyecto final de la materia **Metodologías de Desarrollo de Software**. Su propósito es concentrar la documentación, evidencias y productos de trabajo utilizados para el análisis, diseño, planeación, construcción y validación del sistema.

## Contenido

El repositorio incluye los siguientes documentos y recursos:

1. Matriz de Rastreabilidad del Proyecto  
2. Análisis de Riesgos  
3. Sprint Backlogs completos  
4. Casos de Uso en su totalidad  
5. Prototipo completo  
6. Diagrama de Clases  
7. Base de Datos customizada  
8. Diagramas de Flujo de Datos  
9. Aplicación construida  
10. Firmas de Aceptación de la Propuesta de Negocio  
11. Carta de Aceptación de la primera entrega  
12. Minutas de Junta del Proyecto  
13. Propuestas de Mejora de cada entrega  

## Descripción

Los artefactos aquí presentados documentan el proceso de desarrollo del **Sistema de Biblioteca Inteligente UDEM con Chatbot Integrado**, incluyendo la definición de requerimientos, el seguimiento de actividades por sprint, la identificación de riesgos, el modelado del sistema, el diseño de la base de datos, la construcción del prototipo y la evidencia de aceptación de entregas.

## Metodología

El proyecto fue desarrollado bajo un enfoque ágil basado en **SCRUM**, organizando el trabajo en sprints y documentando los avances mediante backlogs, minutas, propuestas de mejora y artefactos técnicos.

## Acceso

Este repositorio cuenta con permisos de visualización habilitados para su revisión académica.
