# Biblioteca App

Frontend Angular del sistema Biblioteca UDEM.

## Stack

- Angular 21
- Standalone components
- Angular Router
- SCSS
- Mock data en español para demo visual

## Pantallas incluidas

- `/login`
- `/dashboard`
- `/usuarios`

## Instalar dependencias

```powershell
npm.cmd install
```

## Desarrollo

```powershell
npm.cmd start
```

Abre:

- `http://localhost:4200`

## Build

```powershell
npm.cmd run build
```

Salida de build:

- `dist/biblioteca-app`

## Notas

- El frontend esta preparado para conectarse despues con la API de `backend/Biblioteca.API`.
- Las vistas actuales usan mock data para conservar fidelidad visual y velocidad de iteracion.
